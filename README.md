# meta-analyses of LV apical aneurysms in HCM

LVA is “A discrete, thin walled dyskinetic or akinetic segment of the most
distal portion of the LV”

Subgroup analyses:
* Classification according to size
* Impact of imaging modality
* Phenotype
* Geography

Outomes of interest:
* Prevalence
* SCD
* NSVT
* AF
* LVA
